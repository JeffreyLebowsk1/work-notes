# 🔁 Recurring Tasks

A master list of everything that happens on a regular schedule. Check these off each cycle and use them as a safety net so nothing slips through.

---

## 📅 Daily

- [ ] Check and respond to student/staff emails
- [ ] Process any pending transcript requests
- [ ] Update daily log

---

## 📅 Weekly

- [ ] Review open action items and issues
- [ ] Follow up on pending residency appeals
- [ ] Check graduation application queue (during active season)

---

## 📅 Monthly

- [ ] Run enrollment verification reports
- [ ] Review and update contacts directory
- [ ] Check for any policy or system updates
- [ ] Archive completed meeting notes

---

## 📅 Start of Each Semester

- [ ] Update academic calendar with new semester dates
- [ ] Open graduation applications
- [ ] Send graduation application announcement to students
- [ ] Confirm commencement ceremony date and venue
- [ ] Begin cap & gown vendor coordination
- [ ] Update residency classification for new admits
- [ ] Review and update FAQ with any new common questions
- [ ] Review and update email templates

---

## 📅 Mid-Semester

- [ ] Send graduation application deadline reminder
- [ ] Begin preliminary degree audits
- [ ] Coordinate with advisors on graduation candidates
- [ ] Draft commencement program

---

## 📅 End of Semester

- [ ] Close graduation applications
- [ ] Finalize graduation candidate list
- [ ] Submit commencement program for printing
- [ ] Confirm diploma order
- [ ] Run final degree audits
- [ ] Submit graduation list to Provost / Academic Affairs
- [ ] Conduct commencement ceremony
- [ ] Update student records with graduation status
- [ ] Confirm diplomas mailed / available for pickup
- [ ] Archive semester graduation files
- [ ] Document lessons learned

---

## 📅 Annually

- [ ] IPEDS reporting
- [ ] Review and update all office procedures
- [ ] Review FERPA compliance
- [ ] Renew vendor contracts (cap & gown, diploma, transcript service)
- [ ] Update contacts directory
- [ ] Archive prior year files
