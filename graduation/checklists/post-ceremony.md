# Post-Ceremony Checklist

## Immediately After
- [ ] Update graduation candidate list with any last-minute changes
- [ ] Confirm diploma order with vendor
- [ ] Send congratulatory communications to graduates

## Within 1 Week
- [ ] Submit final graduation list to records
- [ ] Post ceremony photos (if applicable)
- [ ] Collect and file ceremony materials

## Within 1 Month
- [ ] Confirm diplomas mailed / available for pickup
- [ ] Update student records with graduation status
- [ ] Archive ceremony files and notes
- [ ] Document lessons learned for next cycle
