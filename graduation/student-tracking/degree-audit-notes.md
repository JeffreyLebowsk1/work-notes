# Degree Audit Notes

## Audit Process Overview

A degree audit verifies that a student has fulfilled all academic requirements for their degree program. This is done before the graduation candidate list is finalized.

1. Pull the preliminary graduation applicant list from SIS.
2. Run a degree audit report for each applicant in the academic advising / audit system (e.g., Degree Works, Ellucian, etc.).
3. Identify students with:
   - **Completed** — all requirements met ✅
   - **In Progress** — requirements being met in the current term (conditionally approved) ⚠️
   - **Incomplete / Deficient** — one or more requirements not met ❌
4. Flag and contact advisors for all students with unmet requirements.
5. Allow time for students/advisors to resolve issues (course substitutions, petitions, etc.).
6. Conduct a final audit pass after the resolve period.
7. Compile the **final approved graduation list**.
8. Submit to the Registrar/Records for official graduation posting.

---

## Common Issues

| Issue | What to Do |
|---|---|
| Missing course in a required sequence | Consult with academic advisor; may require substitution petition |
| Transfer credit not applied to a requirement | Request departmental review for transfer equivalency |
| Student in their last term, course in progress | Conditionally approve; follow up after grades post |
| Grade not yet posted (incomplete "I" grade) | Cannot graduate with an incomplete unless resolved; notify student and advisor |
| GPA below graduation minimum | Cannot approve; notify student and advisor of options |
| Student applied for wrong degree/program | Confirm intended program with student; update application in SIS |
| Duplicate credit issues | Consult with Registrar/advisor to determine which credit counts |
| Honors designation question | Verify GPA and hours completed meet honors threshold |

## Working with Advisors

- Send advisors the preliminary candidate list as early as possible — give them adequate review time.
- Use a shared template or spreadsheet to communicate audit status clearly (see [Student Tracking data folder](./data/)).
- Set a **clear deadline** for advisor responses.
- Follow up on non-responses before the deadline.
- If an advisor approves a student with an exception, get it in writing (email confirmation is sufficient).

### Advisor Communication Timeline
1. **~3 months out** — Send preliminary list; ask advisors to flag any concerns.
2. **~2 months out** — Send formal audit results; request review of deficient cases.
3. **Deadline** — Receive advisor sign-offs on all conditionally approved or resolved cases.
4. **~1 month out** — Final confirmed list to Registrar.
