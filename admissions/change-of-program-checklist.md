# 🔄 Change of Program — Processing Checklist

Use this checklist each time a student submits a request to change their program of study.

---

## Required Steps

- [ ] **Verify Catalog of Record:** Ensure the student is moved to the catalog version current at the time of the change.
- [ ] **Form Completion:** Ensure the "Change of Program" form is signed and filed.
- [ ] **Financial Aid Review:** Verify the new program is aid-eligible.
- [ ] **Self-Service Update:** Update the success plan to reflect the new requirements.
