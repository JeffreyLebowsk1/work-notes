# Curriculum Course Section Identification & Section Number Assignments

> **Effective:** 2022SP  
> **Last updated by:** Jamie Childress, 07/2021

---

## Distance Education Section Prefixes

| Section Code | Format | Description |
|---|---|---|
| LN1 | Asynchronous (ASY) | Totally online, no set meeting times |
| LC1 | Synchronous (SYN) | Collaborate course, online, with set meeting time |
| LM1 | Synchronous (SYN) | Blended — 50% or less online; will have set meetings, rest collaborate/online or both |
| LJ1 | Synchronous (SYN) | Hybrid — 50% or more online; will have set meetings, rest collaborate/online or both |

---

## Section Number Structure

Most section numbers contain **three characters**. Short (mini) semesters may contain up to **five characters**.

- **Day Classes:** M–F, beginning **prior to 5:00 p.m.**
- **Evening Classes:** M–TH, beginning **at or after 5:00 p.m.**

### Second (Middle) Character — Class Type

| Code | Meaning |
|---|---|
| B | Blended (formerly M/web-assisted) — 50% or less online |
| C | Collaborate/Web-conference |
| E | Early College |
| H | Hybrid (formerly High school/CCP) — 51% or more online |
| K | K12 — High school class (new for CCP) |
| L | CCP (online) |
| N | Offered via the Internet (distance education) |
| V | Teleconferencing using Smartroom technology |
| W | Weekend class |
| 0 (zero) | Traditional day or evening class |

---

### Third (Right-Most) Character — Time of Day

| Code | Meaning |
|---|---|
| 1–9 | Day or weekend class beginning **prior to 5:00 p.m.** |
| A–Z | Evening or weekend class beginning **at or after 5:00 p.m.** |

---

## Special Cases

- **Eight-week mini semesters** use `181` or `281` as the last three characters:
  - `1` in the 4th character position = first 8-week offering
  - `2` in the 4th character position = second 8-week offering
- **Two-character sections** denote either independent study (IS) classes or classes offered at the Harnett Correctional Institute (`01`).
