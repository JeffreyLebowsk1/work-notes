# 📋 Course Section Number Reference

> **Reference only.** Last updated by Jamie Childress 07/2021.
> Effective for section number assignments: **2022SP** and forward.

---

## Distance Education / Online Course Types

| Section Code | Type | Description |
|---|---|---|
| **LN1** | Totally Online (Async) | No set meeting times — **ASY** (asynchronous) |
| **LC1** | Collaborate / Online (Sync) | Online with set meeting times — **SYN** (synchronous) |
| **LM1** | Blended | 50% or less online; will have set meeting times; remainder is collaborate, online, or both — **SYN** |
| **LJ1** | Hybrid | 50% or more online; will have set meeting times; remainder is collaborate, online, or both — **SYN** |

---

## Section Number Assignment Structure

Most section numbers contain **three characters**. Short semesters may contain up to **five characters**.

### Schedule Type

| Time | Definition |
|---|---|
| **Day Classes** | Monday–Friday, beginning **prior to 5:00 p.m.** |
| **Evening Classes** | Monday–Thursday, beginning **at or after 5:00 p.m.** |

---

### Second (Middle) Character — Course Delivery Method

| Code | Delivery Method |
|---|---|
| B | Blended (formerly M/web-assisted) — 50% or less online |
| C | Collaborate / Web-conference |
| E | Early College |
| H | Hybrid (formerly High school/CCP) — 51% or more online |
| K | K12 — High school class (new for CCP) |
| L | CCP (online) |
| N | Offered via the Internet (distance education) |
| V | Teleconferencing using Smartroom technology |
| W | Weekend class |
| 0 (zero) | Traditional day or evening class |

---

### Third (Right-Most) Character — Section Sequence / Time of Day

| Code | Meaning |
|---|---|
| 1–9 | Day or weekend class beginning **prior to 5:00 p.m.** |
| A–Z | Evening or weekend class beginning **at or after 5:00 p.m.** |

> **Eight-week mini semesters** use **181** or **281** depending on first or second week offering.
> - `1` designates first 8-week session
> - `2` designates second 8-week session

**Two-character sections** denote either independent study (IS) classes or classes offered at the Harnett Correctional Institute (01).
