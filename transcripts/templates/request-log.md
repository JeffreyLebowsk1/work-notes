## Request Type Reference

| Type | Description |
|---|---|
| **Official** | Sealed/certified transcript sent to a third party or student-sealed |
| **Unofficial** | Unsealed copy for student's personal use |
| **Rush** | Expedited processing requested |
| **Electronic** | Delivered via e-transcript service (Parchment, NSC, etc.) |

## Notes

- Log holds a **record of all outgoing transcript requests**.
- Archive completed logs monthly.
- Do not log student SSN or full DOB in this file — use Student ID only.
- Flag any denied requests (holds, incomplete info) with a note.
