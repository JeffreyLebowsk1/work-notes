# 📋 Foreign Transcript Receival Form

*Source: `assets/documents/Foreign Transcript Receival Form.pdf`*  
*Fillable PDF: print from `assets/documents/` or request from the Registrar's Office.*

Used to document the receipt and processing of foreign (international) transcripts submitted by students applying with international academic credentials.

## Form Fields

### Student Information
- **Full Name**
- **Student ID**

### Student Records & Registrar Staff

| Field | Description |
|---|---|
| Date Received | Date the foreign transcript was received |
| Received By | Name of SRR staff member who received it |
| Method of Receipt | ☐ Email  ☐ Third-Party Download  ☐ Hard Copy Sealed  ☐ Hard Copy Unsealed  ☐ Email Verification  ☐ Envelope |
| Other Documents Attached | Any supporting documents received with the transcript |
| SRR Staff Comments | Notes from Student Records & Registrar staff |
| Registrar Comments | Notes from the Registrar |

### Dean of Enrollment

| Field | Description |
|---|---|
| Date Received | Date received by Dean of Enrollment |
| Signature | Dean of Enrollment signature |
| Date Processed | Date the transcript was fully processed |

## Notes on Use

- Use this form whenever a foreign transcript arrives via any method — log it immediately.
- Hard copy sealed transcripts must remain sealed until the Registrar reviews them.
- Attach this form to the physical transcript or note in the student's file.
- For credential evaluation procedures, see [processes.md](./processes.md).
