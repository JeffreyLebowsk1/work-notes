# Residency & Tuition Procedures

## Determining Residency Status

### For New Students (at Admission)
1. Student completes a **Residency Questionnaire** as part of the application process.
2. Registrar's Office reviews questionnaire responses and any submitted documentation.
3. Apply classification criteria from [Residency & Tuition Policies](./policies.md):
   - If all criteria are clearly met → classify as **in-state**.
   - If criteria are not met or documentation is incomplete → classify as **out-of-state** (default for ambiguous cases).
   - If visa status is non-immigrant → classify as **non-resident** (or international rate).
4. Enter classification in SIS.
5. Notify student of their classification via email/letter.

### For Continuing Students (Reclassification Request)
1. Student submits a Reclassification Request Form with supporting documentation.
2. Verify documentation against the criteria in [Policies](./policies.md).
3. Confirm the 12-month domicile period has been met **before the start of the requested term**.
4. Update SIS if reclassification is approved.
5. Notify student of decision in writing.
6. If denied, advise the student of the appeal process.

## Documentation Required

### For In-State Classification
At least **three** of the following:
- [ ] State driver's license (issued 12+ months before term start)
- [ ] State vehicle registration
- [ ] State voter registration
- [ ] State income tax return (most recent)
- [ ] Residential lease or mortgage (in-state, covering 12+ months prior)
- [ ] Employer letter confirming in-state employment (non-school employment)
- [ ] Bank statements with in-state address (12+ months)

### For Military / Veteran Rate
- [ ] Copy of military orders (active duty) or DD-214 (veteran)
- [ ] Spouse/dependent: copy of sponsor's orders and proof of relationship

## Common Scenarios

| Scenario | How to Handle |
|---|---|
| Student moved to state to attend school | Out-of-state; does not qualify on domicile alone |
| Dependent student — parents in-state, student attends out-of-state school | In-state based on parental domicile |
| Student turned 24, now financially independent | Evaluate on student's own domicile from date of independence |
| Military student stationed in state | In-state per military provisions; request orders |
| F-1 visa student | Non-resident/international rate; non-immigrant visa disqualifies |
| Student married an in-state resident | Does not automatically confer residency; evaluate domicile independently |
| Student can't produce all 3 documents | Review what's available; classify out-of-state if insufficient; advise on appeal |
