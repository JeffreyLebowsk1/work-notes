# 📸 Screenshot Capture Checklist

Screenshots of Colleague/Datatel screens referenced in staff procedure notes.

**How to use:**
1. Open Colleague and navigate to the screen listed below.
2. Follow the "What to capture" description.
3. Save the file with the exact filename shown into the correct folder.
4. Check off the item when done — the placeholder in the note will automatically become a live image.
